# sbd-nextjs-landing-page

The **Landing Page** serves as the public face of the Second Brain Database project. It showcases the platform's features, pricing, and capabilities to potential users.

## Features

-   **Modern Design**: Sleek and responsive layout using Tailwind CSS.
-   **Interactive Elements**: Animations and transitions powered by Framer Motion.
-   **Diagrams**: Integrated Mermaid diagrams for architecture visualization.
-   **Drag & Drop**: Interactive components using `@dnd-kit`.

## Tech Stack

-   **Framework**: [Next.js 15](https://nextjs.org/)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS, Tailwind Animate
-   **Animations**: Framer Motion
-   **Icons**: Phosphor Icons, Lucide React
-   **State Management**: Zustand

## Prerequisites

-   Node.js 18+
-   pnpm (recommended) or npm/yarn

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

-   `npm run dev`: Run the development server.
-   `npm run build`: Build the application.
-   `npm start`: Start the production server.
-   `npm run lint`: Run ESLint.
